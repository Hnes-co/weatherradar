# Weatherradar

## Asennusohjeet 

```bash
git clone https://github.com/Hnes-co/weatherradar.git

cd weatherradar

npm install

npm start
```

## Api keyn lisäys

- Sovellus käyttää openweathermap API:a
- Luo src-kansioon 'api-connection.json' - tiedosto ja lisää oma api key:
```bash
{
    "apiKey": "apikey tähän"
}
```
