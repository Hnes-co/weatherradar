# Weatherradar

## How to install 

```bash
git clone https://github.com/Hnes-co/weatherradar.git

cd weatherradar

npm install

npm start
```

## Add your API key

- This project uses openweathermap API
- Create 'api-connection.json' - file in /src and add your api key:
```bash
{
    "apiKey": "apikey tähän"
}
```
